# Python - Functions & Modularity

This project introduces the basics of functions and modular programming in Python. It shows how to write small reusable pieces of code, import functions from other files, and keep programmes organised instead of putting everything in one place.

## Key concepts

- Defining functions with `def`
- Returning values with `return`
- Importing functions from another file
- Using `if __name__ == "__main__":`
- Writing cleaner and more reusable code
